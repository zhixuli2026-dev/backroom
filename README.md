# 后室 (The Backrooms) - 3D 恐怖探索游戏

一款基于网络都市传说"后室"的 3D 第一人称恐怖探索游戏，使用 Three.js 和 TypeScript 开发。

![游戏截图](./screenshot.png)

## 🎮 游戏简介

你意外地从现实世界"穿模"进入了后室 - 一个充满单调黄色墙壁、潮湿地毯和嗡嗡作响的荧光灯的无尽迷宫。你必须收集散落的出口碎片，躲避潜伏在走廊中的恐怖实体，找到逃离这个噩梦般空间的方法。

## ✨ 游戏特性

### 核心玩法
- 🎯 **收集碎片**：在迷宫中找到 4 个出口碎片
- 🚪 **逃离后室**：收集所有碎片后通过出口门
- 👹 **躲避怪物**：2 只巡逻的恐怖实体会追逐并捕获你
- 🏃 **生存逃脱**：在被抓住前成功逃离

### 视觉效果
- 🏢 **高度还原的后室场景**
  - 泛黄的墙壁材质（带污渍和水渍）
  - 棕色地毯纹理（纤维和污渍细节）
  - 米色吊顶（格子纹理）
- 💡 **昏暗的荧光灯光照**
  - 昏黄的灯光效果
  - 随机闪烁的荧光灯
  - 动态阴影系统
- 🌫️ **压抑的氛围**
  - 指数雾效果
  - 暗色调色彩映射
  - 幽闭恐惧感设计

### 交互系统
- 🕹️ **流畅的第一人称控制**
  - WASD 移动
  - 鼠标视角控制
  - Shift 奔跑
  - 碰撞检测和滑动
- ✋ **交互提示系统**
  - 靠近物品时显示提示
  - E 键收集物品
  - 实时 UI 反馈

### 怪物 AI
- 🤖 **智能行为系统**
  - **巡逻模式**：沿预设路径巡逻
  - **追逐模式**：发现玩家后快速追逐
  - **搜索模式**：失去目标后原地搜索 30 秒
- 👁️ **检测范围**：10 米检测半径
- 🏃 **速度变化**：巡逻时慢速，追逐时快速
- 🔴 **视觉设计**：黑色身躯 + 红色发光眼睛

## 🎨 技术栈

- **前端框架**: Vite
- **开发语言**: TypeScript
- **3D 引擎**: Three.js
- **碰撞检测**: 自定义 AABB 系统

## 🚀 快速开始

### 前置要求
- Node.js 18+
- npm 或 yarn

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/zhixuli2026-dev/backroom.git
cd backroom
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 打开浏览器访问 `http://localhost:5173`

### 构建生产版本

```bash
npm run build
```

构建后的文件将输出到 `dist/` 目录。

## 🗺️ 游戏地图

游戏包含 5 个相连的后室房间，呈 L 形布局：

```
[房间2-碎片2] - [房间1-碎片1] - [房间0-出生点]
                                      |
                                 [房间3-碎片3]
                                      |
                                 [房间4-碎片4+出口门]
```

- **出生点**：房间 0（中心位置）
- **碎片位置**：分散在 4 个不同房间
- **怪物巡逻**：
  - 怪物 1：东侧走廊（房间 1-2）
  - 怪物 2：南侧走廊（房间 3-4）
- **出口门**：最远的房间 4

## 🎯 游戏目标与通关条件

### 胜利条件
1. 收集所有 4 个发光的青色碎片
2. 前往最后一个房间
3. 与解锁的绿色出口门交互（按 E 键）

### 失败条件
- 被任意一只怪物抓住（距离小于 1 米）

## 🔧 技术细节

### 性能优化
- 限制渲染像素比为 2
- 阴影贴图分辨率 512x512
- 指数雾效果减少渲染距离
- 程序化纹理生成（无需加载外部图片）

### 碰撞检测
- AABB（轴对齐包围盒）碰撞检测
- 碰撞解析算法（推出最小距离）
- 玩家碰撞体：0.8m x 1.7m x 0.8m

### AI 系统
- 有限状态机（FSM）：巡逻 → 追逐 → 搜索
- 路径点导航系统
- 平滑旋转插值
- 距离检测触发追逐

## 🎨 美术风格

游戏完全采用程序化生成的纹理和材质，无需外部图片资源：

- **墙壁**：Canvas 绘制的泛黄纹理 + 随机污渍和水渍
- **地毯**：Canvas 绘制的棕色纤维纹理 + 污渍
- **天花板**：Canvas 绘制的米色吊顶格子
- **光照**：昏黄的点光源 + 环境光 + 半球光
